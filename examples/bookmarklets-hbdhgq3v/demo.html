Bookmarklet Example

These links, like any HTML links, can be bookmarked in the browser. When you click on the bookmark, the Javascript code in the href attribute of these link tags will execute in the context of the current window object. This means you can use this bookmark to modify any page you happen to be viewing.

To see this work, drag the link from this page to your bookmarks bar, then navigate to any Web page and click the bookmark.
