HTML5 Form Validation - Patterns and Regular Expressions

Notice the pattern attribute in the input tags in the raw HTML panel. Type something in the fields, and then submit the form to see HTML5 validation messages.

Using the pattern "\d+" means "one or more digits" are acceptable.


Using the pattern ".+@.+\.edu" breaks down into several segments:

  • .+ -- one or more characters
  • @ -- then an '@' sign
  • .+ -- then one or more characters again
  • \. -- then a dot
  • edu -- then the letters 'edu'


The pattern "\d{3}-\d{3}-\d{4}" breaks down into

  • three digits followed by a hyphen \d{3}-
  • another three digits followed by a hyphen\d{3}-
  • and then four digits \d{4}
