Function Naming
  • Function names can begin with $, _ or a letter (but not a digit)
  • Function names can contain letters or numbers or some punctuation
  • Javascript operators are not allowed
  • While unicode characters are technically allowed, it's safest to stick to standard ASCII, and avoid punctuation marks other than _ or $
