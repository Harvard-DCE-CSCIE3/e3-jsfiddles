Basic Object Properties - 3

'dot' notation and 'bracket' notation to access object properties

  • If keys conform to rules for variable names, each notation is equivalent
  • Keys can be numeric, but then only bracket notation can be used to access them
  • Keys can also be arbitrary strings with special characters in them, but then you need quotes in the object literal, and bracket notation to access
  • Keys can also be referened by variables, and you can use bracket notation to access
