Arrays & Looping

4 Tasks for this Practice Set:

  1. Make an Array of strings and assign it to a variable. You may make the strings a list of anything you like and use a variable name of your choice.
  2. Use one of the methods of the Array object to insert a new element to the beginning of your array. Looking through the methods on the Array reference page at W3Schools may be helpful.
  3. Using the Array you made in the previous steps, iterate over its elements and write each one to the console.
  4. Loop through the integers from 1 to 100 and write each to the console only if it is odd.
