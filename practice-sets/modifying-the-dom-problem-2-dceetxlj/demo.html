Practice Set #2, Week 8, Modifying the DOM

We're going to practice the steps of manually creating an adding a DOM element to a page. To do this, we'll create a DIV with id="myNewDiv" that contains some text, and attach it to the div already on this page with id="addNewOneHere". If you've done it right, it will appear with a red background.

Hint: There's five steps here, as shown in the lessons, particularly lesson 3 section 2, the TOC example in Video 8.6, and the readings on javascript.info. You'll have to:

  1. create a 'DIV' element node and assign it to a variable
  2. using setAttribute(), give it an ID attribute of "myNewDiv"
  3. create a text node with some text of your choosing, and assign it to a variable
  4. append you text node to your element
  5. append your new element to the existing div which has id="addNewOneHere"

This is the div where you'll add your new element using Javascript.
