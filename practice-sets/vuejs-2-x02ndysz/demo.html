VueJS Exercise: Create the data and a button action

In this exercise, we've created a skeletal Vue object in JavaScript and a template in the HTML that displays a generic button.

Your tasks are to:

  • Create some data for the object, and display this data in the HTML using the Vue template language. It can be anything: a single value or simple object, or a list
  • Create a method that does something with the data, and connect it to the button, so that when the button is clicked, the data, or its display changes. For example, you might use the button to change a property, add an item to a list, remove or hide an item, or change the display style of the list.
  • If the change is just to the display style, be sure to use a Vue property and directive (see Class and Style Bindings) to manifest the change.
