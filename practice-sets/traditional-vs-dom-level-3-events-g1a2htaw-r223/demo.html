Practice Set, Week 9, Traditional vs. DOM Level 3 Events

This page contains a button, and code written using the traditional method of attaching events to DOM elements. We'd like to be sure that you're comfortable with the syntax for attaching events, especially the DOM Level 3 method (e.g. using addEventListener()).

Your task is to rewrite the event handler so that it uses the DOM Level 3 method of attaching events (e.g. using addEventListener() rather than the ondblclick property)

The functionality should remain identical. Double-clicking the button will alternate its text color between red and blue.
